## üßæ Pandas Drop Cheat Sheet
### üîç Inspect Before Dropping
```python
df.info()                         # Overview of columns & nulls
df.isnull().sum()                # Count of missing values per column
df.duplicated().sum()            # Count of duplicated rows
df.describe(include='all')       # Summary of all columns
```
### üìâ Drop Rows with Missing Values
# Drop rows only if 'email' or 'user_id' is missing
```python
df = df.dropna(subset=['email', 'user_id'])
```
# Drop all rows with any missing value (use with care)
```python
df = df.dropna()
```
# Drop rows where all values are missing
```python
df = df.dropna(how='all')
```
### üìâ Drop Columns with Too Many Nulls
# Drop columns where more than 80% of values are missing
```python
threshold = 0.8
cols_to_drop = df.columns[df.isnull().mean() > threshold]
df = df.drop(columns=cols_to_drop)
```
### üßπ Drop Duplicate Rows
# Drop fully duplicated rows
```python
df = df.drop_duplicates()
```
# Drop duplicates based on specific columns
```python
df = df.drop_duplicates(subset=['name', 'email'], keep='first')  # or 'last', or False
```
### üß± Drop Specific Columns or Rows
# Drop a column
```python
df = df.drop(columns=['unnecessary_column'])
```
# Drop multiple columns
```python
df = df.drop(columns=['col1', 'col2'])
```
# Drop a row by index
```python
df = df.drop(index=3)
```
# Drop multiple rows by index
```python
df = df.drop(index=[0, 2, 4])
```
### üõë Use inplace=True only when sure
# Avoid until you're confident
```python
df.drop(columns=['temp'], inplace=True)
```
### ‚ôªÔ∏è Reset Index After Dropping Rows
```python
df = df.reset_index(drop=True)
```
### üßæ Log Your Drops (Good for pipelines)
```python
before = df.shape[0]
df = df.dropna()
after = df.shape[0]
print(f"Dropped {before - after} rows with missing values.")
```
