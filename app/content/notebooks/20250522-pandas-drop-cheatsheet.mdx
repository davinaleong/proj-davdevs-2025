### ğŸ“‰ Drop Rows with Missing ValuesÂ¶

# Drop rows only if 'email' or 'user_id' is missing

# Drop all rows with any missing value (use with care)

# Drop rows where all values are missing

### ğŸ“‰ Drop Columns with Too Many NullsÂ¶

df = df.drop(columns=cols_to_drop)

### ğŸ§¹ Drop Duplicate RowsÂ¶

# Drop fully duplicated rows

# Drop duplicates based on specific columns

### ğŸ§± Drop Specific Columns or RowsÂ¶

# Drop a column
df = df.drop(columns=['unnecessary_column'])

# Drop multiple columns
df = df.drop(columns=['col1', 'col2'])

# Drop a row by index
df = df.drop(index=3)

# Drop multiple rows by index
df = df.drop(index=[0, 2, 4])

### ğŸ›‘ Use inplace=True only when sureÂ¶

# Avoid until you're confident

### â™»ï¸ Reset Index After Dropping RowsÂ¶
