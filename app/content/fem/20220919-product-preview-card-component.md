# Product Preview Card Component

An elegant e-commerce product card showcasing perfume with pricing information, product details, and call-to-action button. This Frontend Mentor challenge focuses on commercial product presentation.

## Key Features

- **Product Showcase**: High-quality image presentation
- **Pricing Display**: Original and sale price comparison
- **Product Details**: Category, name, and description
- **Add to Cart**: Prominent call-to-action button
- **Responsive Images**: Optimized for different screen sizes

## Technologies Used

- HTML5
- CSS3
- Flexbox layout
- Responsive images
- Custom button styling

## E-commerce Elements

- Product category labeling
- Pricing typography hierarchy
- Product description formatting
- Call-to-action button design
- Mobile-optimized layout

## Design Considerations

- Commercial product presentation
- Price comparison styling
- Button accessibility and touch targets
- Image optimization for web
- Cross-device consistency
