# Crowdfunding Product Page

A comprehensive crowdfunding landing page featuring interactive pledge selection, modal dialogs, and dynamic progress tracking. This Frontend Mentor challenge showcases complex user interactions and state management.

## Key Features

- **Interactive Pledges**: Multiple backing options with selection
- **Progress Tracking**: Dynamic funding progress visualization
- **Modal Dialogs**: Pledge selection and success modals
- **Bookmark Function**: Save project functionality
- **Responsive Design**: Full mobile and desktop optimization

## Technologies Used

- HTML5
- CSS3 (Grid & Flexbox)
- JavaScript (ES6+)
- Modal components
- State management

## Interactive Elements

- **Pledge Selection**: Multiple reward tier options
- **Modal System**: Overlay dialogs for user interactions
- **Progress Bars**: Visual funding progress indicators
- **Bookmark Toggle**: Save/unsave project functionality
- **Form Validation**: Pledge amount validation

## Advanced Features

- Dynamic content updates
- Interactive pledge calculations
- Modal state management
- Progress percentage calculations
- Responsive modal design

## Implementation Highlights

- Complex JavaScript architecture
- Event-driven programming
- Accessible modal patterns
- Mobile-optimized interactions
- Progressive enhancement
