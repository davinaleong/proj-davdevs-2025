# Stats Preview Card Component

A compelling statistics preview card featuring image overlay effects and clean data presentation. This Frontend Mentor challenge demonstrates advanced CSS techniques for image manipulation and responsive layout.

## Key Features

- **Image Overlay**: Purple tint overlay effect on hero image
- **Statistics Display**: Clean presentation of key metrics
- **Responsive Layout**: Desktop and mobile optimized layouts
- **Typography Hierarchy**: Clear distinction between headings and data
- **CSS Blend Modes**: Advanced image styling techniques

## Technologies Used

- HTML5
- CSS3
- CSS blend modes and filters
- Flexbox layout
- Media queries

## Design Techniques

- CSS blend modes for image overlay
- Responsive image handling
- Statistics grid layout
- Color theory application
- Typography optimization

## Implementation Details

- Cross-browser image overlay compatibility
- Mobile-first responsive approach
- Clean semantic HTML structure
- Optimized image loading
- Accessibility considerations
