# NFT Preview Card Component

A beautifully crafted NFT preview card component built as part of a Frontend Mentor challenge. This project focuses on creating an elegant card design with interactive hover effects and clean typography.

## Key Features

- **Hover Effects**: Smooth image overlay and interactive states
- **Typography**: Proper font hierarchy and readable text styling
- **Card Layout**: Well-structured component design
- **Responsive**: Adapts to different screen sizes
- **Accessibility**: Semantic HTML and proper contrast ratios

## Technologies Used

- HTML5
- CSS3
- CSS Transitions
- Flexbox Layout
- Custom CSS Properties

## Design Elements

- Clean card-based layout
- Elegant hover states on images and links
- Proper spacing and typography
- Color contrast optimization
- Mobile-responsive design
